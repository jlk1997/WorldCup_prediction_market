# 数字藏品合规说明

## 定位

「球星收藏册」中的卡牌为**平台内虚拟数字藏品**，用于娱乐收集、养成与炫耀，**不具备任何金钱价值**。

## 必须遵守

1. **不可交易**：卡牌、碎片不可在用户间转赠、交易或出售。
2. **不可提现**：数字藏品不可兑换现金、不可兑换真实商品（除平台内虚拟奖励外）。
3. **不可金融化**：禁止宣传「投资价值」「保值」「锚定金钱」等金融属性。
4. **获得方式**：仅通过玩法行为获得（竞猜猜中、签到里程碑、比赛日动员、邀请里程碑、碎片合成），**不做付费随机盲盒**。
5. **与比赛结果的关系**：「高光印记」为荣誉成就记录（进球/助攻/完赛），非金融锚定。
6. **消耗方式**：碎片合成、升星消耗平台内「可用积分」与碎片，不影响「累计积分」冲榜。
