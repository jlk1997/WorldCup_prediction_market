# 数字藏品 / 足球数字资产 合规说明

## 定位

「球星收藏册 / 球星卡资产」为**平台内虚拟数字藏品**，用于娱乐收集、养成、玩法赋能与站内流通。**不具备任何金钱价值，不可提现，不可用人民币计价二级交易。**

## 一、计价与货币（合规命门）

1. **二级流通一律以「可用积分」计价**，可用积分为站内虚拟积分，**无现金价值、不可提现、不可兑换人民币**。
2. **人民币只进不出**：人民币仅用于一级首发打新与球迷币充值（虚拟道具购买），**严禁**任何形式的卡牌人民币二级买卖、回购套现、提现。
3. **禁止金融化话术**：全程不得出现「投资」「升值」「保值」「理财」「锚定金钱」等表述；行情仅作站内收藏体验展示，并固定标注免责声明。

## 二、流通规则（合规边界内）

1. **实名前置**：用户须完成实名认证后，方可进行转赠与交易行挂牌/购买（反洗钱、反代充、防未成年人）。
2. **冷却持有期**：新获得卡牌需经过冷却期（默认 7 天）后方可流通，抑制炒作与刷量。
3. **转赠限频**：实名用户间转赠设日/季次数与额度上限。
4. **交易行（积分计价）**：用户可挂牌一口价或发起积分英式竞拍；成交按比例收取**积分手续费**（积分 sink，回收通胀）。
5. **官方回购 / 地板价**：平台按稀有度提供**积分**保底回购，给资产价格底与退出路径——回购仅返还积分，**不返还人民币**。
6. **链上结算**：成交/转赠经 AVATA **托管账户间**转移真实文昌链 NFT；服务端先校验全部合规规则（实名/冷却/限频/锁定态）再调链，平台完全可控。

## 三、获得方式

- 玩法掉落（竞猜猜中、签到里程碑、比赛日动员、邀请里程碑）
- 碎片合成 / 升星（确定性消耗，非随机盲盒）
- 手册等级奖励（确定性发放）
- 一级首发打新（限量发行，预约/白名单/抽签，人民币或球迷币入场）
- 交易行收购、用户转赠

## 四、持有玩法 / 赋能

卡牌具备站内实用价值（非金融价值）：质押产被动**积分/球迷币**、竞猜加成、AI 分析折扣、数字阵容计分。所有产出均为站内虚拟权益，不可提现。
