# 积分赛季重置策略

## 原则

- **累计积分 `season_points`**：赛季排行榜用，世界杯周期内持续累计，赛季末归档展示。
- **可用积分 `redeem_points`**：二级交易、兑换装扮用；**不在赛季中清零**，通过 faucet/sink 调节通胀。
- **球迷币 `fan_coins`**：AI/竞猜消耗，与积分独立。

## 沟通话术（对用户）

> 赛季结束后，排行榜将定格并发放称号/奖励；你的可用积分与球星卡收藏 **不会清零**，继续用于站内玩法与兑换。

## 运营调参

- 参考 Admin 经济看板 `/api/asset/admin/economy` 的 inflation_pct
- 积分产出高峰（竞猜、掉卡）与 sink（交易手续费、AI、质押）保持净流入可控
- 重大调整前 7 日站内公告 + 邮件/通知

## 技术备注

- 若未来需要软重置，仅新增 `season_key` 维度排行榜，不迁移历史 redeem_points
- 禁止 RMB 回购积分或球星卡
