# 最后一舞：世界杯2026 — 用户与商业化 PRD

## 1. 产品愿景

**Slogan：** 预测世界杯，赢取球迷荣耀

面向中国球迷的 2026 世界杯数据 + 虚拟竞猜 + AI 参考平台。AI 分析为留存辅助；核心变现与粘性来自 **球迷币竞猜、排行榜、粉丝文化**，支付宝购买虚拟道具（不可提现）。

## 2. 用户故事

| 角色 | 故事 |
|------|------|
| 新访客 | 浏览赛程/Live/球队，无需登录 |
| 新用户 | 邮箱收验证码 → 自动注册 → 获赠 100 球迷币 |
| 活跃球迷 | 每日签到、对未开赛比赛竞猜、查看 AI 参考 |
| 付费用户 | 支付宝购球迷币/赛季通行证，参与高质押竞猜 |
| 排行榜玩家 | 积累赛季积分，争夺日/周/赛季榜与虚拟勋章 |

## 3. 双货币

- **球迷币 fan_coins**：质押竞猜、助威；可支付宝购买；不可提现
- **赛季积分 season_points**：竞猜准确获得；用于排行榜；不可购买

## 4. 竞猜规则

1. 仅 `scheduled` 且开赛前 30 分钟可提交
2. 玩法：`1X2`（主胜/平/客胜）
3. 质押：10–500 球迷币（每日 1 次免费竞猜不耗币）
4. 结算：猜中返还质押 + 积分；猜错扣除质押
5. 积分：主胜/客胜 +30，平局 +20；连胜 streak 额外 +10/场

## 8. 球迷身份与氛围玩法

### 8.1 球迷档案（软引导）

- 登录后可进入 `/onboarding` 三步向导：主队（必选）→ 副队（可选）→ 最多 3 名球星
- 可跳过；顶栏与球迷中心保留「完善档案」入口
- 未完成档案：允许浏览、签到、每日 1 次免费竞猜；助威与质押竞猜需先完善档案

### 8.2 个性化触达

- `GET /api/profile/recommendations` 聚合主队下一场、副队赛程、球星场次与 CTA
- 赛事大屏优先展示主队比赛；Live「我的球队」Tab；竞猜大厅主队场次置顶高亮
- AI 工作台未选队时预填主队下一场

### 8.3 氛围玩法

| 玩法 | 规则 |
|------|------|
| 助威墙 `/cheer/:matchId` | 5 币/次，+10 助威值；每用户每赛 1 次；仅主/副队 |
| 比赛日签到 | 主队比赛日签到额外 +10 币（共 30） |
| 主队问答 | 每日 1 题，答对 +15 币；未选主队奖励减半 |
| 球迷名片 `/me/card` | 展示主副队、球星、胜率、助威值 |

### 8.4 排行榜增强

- 积分榜（日/周/赛季）
- 主队贡献榜：同队球迷按**军团贡献值**排名（见 §9）
- 主队粉丝榜：各队铁粉人数

### 9.4 玩法模块

- **铁粉军团榜**：同主队球迷按军团贡献排名，展示队内段位
- **双队擂台** `/arena`：赛前 72h 至开赛后两主队球迷贡献对比
- **球星热力榜**：全站 Top 球星 + 个人贡献
- **比赛日动员**：全队贡献达阶梯目标（500/2000/5000）发虚拟称号
- **队内段位**（每周重算）：先锋 Top10% / 主力 Top30% / 替补 Top60% / 新兵

### 9.5 合规

- 界面文案使用「军团贡献 / 热力 / 娱乐排行」
- 无聊天、互评、私信；排行榜仅昵称与数值
- 花币加码标注「虚拟道具，不可提现」
