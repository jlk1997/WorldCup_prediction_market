# 传奇球星视觉素材说明

本目录包含上线用的三位球星动感肖像，采用统一的「世界杯电影海报」画风（暗色球场背景 + 金色轮廓光 + 玫瑰色点缀），供登录页、商城、竞猜大厅与赛事大屏展示。

| 文件 | 球星 |
|------|------|
| `ronaldo.webp` | C罗（卡片/登录，~640px） |
| `messi.webp` | 梅西 |
| `neymar.webp` | 内马尔 |
| `*-backdrop.webp` | 首页大屏背景（~960px，体积更小） |

重新压缩：`python frontend/scripts/optimize-legends.py`（需 `pip install pillow`）

素材为站内原创风格化插画，仅用于「最后一舞」产品 UI 装饰，不构成任何官方代言或授权暗示。
